# FrinX Game Engine - Open Source GTA-like Game Engine
**This project is still in early alpha stage.**

## Try It Now!
### Compiling
Just type:

	scons

### Controlls

	w,s,a,d         move
	LEFT, RIGHT     turn
	ESC             quit
